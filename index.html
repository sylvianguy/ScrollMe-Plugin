Easing Plugin

What Does this Plugin Do?

This is a jQuery plugin to give advanced easing options. This plugin uses another easing plugin from GSGD that enables these advanced easing options. This website will demonstrate 4 out of many different easing effects.

How Does it Work?

Download the following files:

  • jquery.easing.1.3.js
  • scrollMe.js

(Demo: 'easeOutBounce')

Once you have included these files into your own working folders, be sure to include a script tag in your HTML file, right before your closing body tag with the appropriate path to the folder you have stored the plug-in file. See example below. 

  • <script src="jquery.easing.1.3.js"></script> 
    •  
			
  • <script src="scrollMe.js"></script>
    •

    You'll need to give all your button tags a common class name to pass a selector in your click function in the scrollMe.js file. In this example, I have given my button tags a class of "demo". See example below.

    (Demo: 'easeOutQuart')

    You'll also want to give an id for each div/container that each scroll will be animated to and from. I have given each of my containers an id of 'demo' following a number from 1-4 (i.e 'demo1', 'demo2' etc.). Give your button tags 2 data attributes. First attribute: 

    'data-goto'
    to direct the scroll to the next section of your page. Second attribute: 
    'data-easetype'
    to set an easing method if you want to overwrite the easing default of 'easeOutQuad'.

    				
  • <button><div class="demo" data-goto="#demo4"
    • 
				
  • data-easetype="easeInQuint">Demo</button></div>
